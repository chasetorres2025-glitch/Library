# 图书管理系统开发计划

## 环境准备
## 项目初始化

### 后端（SpringBoot）

1. 创建 SpringBoot 3.x 项目结构
2. 配置 Maven 依赖（MyBatis-Plus、Spring Security、Lombok、MySQL 驱动等）
3. 创建数据库连接配置
4. 创建 7 张核心数据表（sys\_user、sys\_role、book\_category、book\_info、book\_stock、book\_borrow、sys\_log）

### 前端（Vue 3）

1. 使用 Vite 创建 Vue 3 项目
2. 安装 Element Plus、Vue Router、Pinia、Axios 等依赖
3. 配置项目基础结构（api、components、views、router、store、utils）

## 核心功能实现

### 后端实现

1. 创建实体类（Entity）、数据访问层（Mapper）、服务层（Service）、控制器（Controller）
2. 实现 Spring Security 权限配置
3. 实现全局异常处理
4. 实现 4 大模块的 CRUD 接口：

   * 系统管理（用户、角色、日志）

   * 图书管理（分类、图书信息、库存）

   * 借阅管理（借阅、归还、记录查询）

   * 前台门户（登录、图书浏览、个人中心）

### 前端实现

1. 封装 Axios 请求拦截器
2. 实现路由守卫和权限控制
3. 开发各功能模块页面和组件
4. 实现响应式布局和交互逻辑

## 测试与优化

1. 后端接口测试
2. 前后端联调
3. 功能测试和修复
